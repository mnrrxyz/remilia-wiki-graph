## How to Update This File

When making changes, add them under `[Unreleased]` in the appropriate category:

- **Added** - New features
- **Changed** - Changes in existing functionality  
- **Deprecated** - Soon-to-be removed features
- **Removed** - Removed features
- **Fixed** - Bug fixes
- **Security** - Security improvements

When releasing a version, move items from `[Unreleased]` to a new version section with a date.

### Example Entry

```markdown
## [0.1.0] - 2024-01-15

### Added
- Basic graph rendering with react-force-graph
- Hover highlighting for nodes and edges
- JSON data loading from remilia_graph_final.json
```
