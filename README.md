# Remilia Wiki Graph

Interactive visualization of the [Remilia Wiki](https://wiki.remilia.org/) conceptual network.

## Overview

Web-based force-directed graph that maps relationships between pages in the Remilia wiki. Built with React + TypeScript, rendered with react-force-graph, hosted on Vercel.

## Stats

- **173** wiki pages
- **2,175** connections between pages
- **284** referenced but missing pages
